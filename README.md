# BASECARRIERBOARDATLAS 
Este diseño se toma como diseño base por la experiencia obtenida en este modelo,se ha minimizado el número de componentes a usar, para que el costo de la placa pueda ser lo más reducido posible.
Cierto es que los componentes usados son de un tamaño mucho menor al modelo inicial, pero la ganancia en sensibilidad de vídeo en es palpable.

La versatilidad de IO-BOARD-ATLAS es muy grande dado que existe la posibilidad de agregar el diseño de los recolocadores, que se veran incrementados con el paso del tiempo, en este caso se vé una pi-pico y el diseño del recolocador de Carlos Palmero.
